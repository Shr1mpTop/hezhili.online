## 🚀 主要项目

这是主要的个人展示网站，采用现代化的技术栈构建：

- **技术栈**: Vue 3 + TypeScript + Tailwind CSS + Vite
- **特性**: 响应式设计、主题切换、动画效果、性能优化
- **访问**: [https://hezhili.online](https://hezhili.online)

详细信息请查看 [`vue-app/README.md`](vue-app/README.md)
